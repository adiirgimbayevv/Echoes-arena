# Echoes Arena

## Tech Stack
- Java
- LibGDX
- Tiled
- Aseprite / Krita
- Gradle

## MVP Scope
- 1 playable hero system
- 5 archetype characters
- 3 weapons
- 3 base elements + 2 combo elements
- 10 story stages
- 5–6 enemy roles
- 2 generals + 1 final boss
- level-up upgrade choices
- meta stat points between stages
- save/load
- HUD + pause + game over

## Done Definition
- player moves, attacks, takes damage
- at least 3 enemy roles work correctly
- wave system works
- one full stage is playable end-to-end
- level-up screen works
- stage result screen gives stat points
- one combo element works
- one boss works
- no critical crashes in 10-minute session

## Art Rules
- one visual style only
- same tile size everywhere
- same outline/shadow logic everywhere
- same color palette family
- readable enemy telegraphs

## Code Rules
- no balance numbers hardcoded inside random classes
- no screen creates other screens directly
- no UI logic inside combat classes
- every new feature must have acceptance criteria
- every new enemy uses factory + strategy + state

## Weekly Milestones
- Week 1–2: core loop
- Week 3–4: waves + enemies + UI
- Week 5–6: progression + weapons + stages
- Week 7–8: bosses + polish + fixes
